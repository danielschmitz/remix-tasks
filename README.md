# Remix Tasks

A simple Tasks manager with

- React + Remix
- Picocss
- Knex
- Vercel deploy ([demo](https://remix-tasks.vercel.app/tasks))

## Install

```
git clone https://github.com/danielschmitz/remix-tasks.git
cd remix-tasks
npm install
npm run createdb
npm run dev
```
